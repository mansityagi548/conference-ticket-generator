# 🎟️ Conference Ticket Generator

A responsive and interactive conference ticket generator built using **HTML, CSS, and JavaScript**.

Users can enter their personal details through a form, and the application dynamically generates a visually styled conference ticket in real time. This project focuses on DOM manipulation, form handling, validation, and responsive UI development.

Although this was my first JavaScript project, it helped me build a strong foundation in frontend development and interactive web applications.
